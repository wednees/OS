# Задание.
Child1 переводит строки в верхний регистр. Child2 убирает все задвоенные пробелы. (Для взаимодействия между процессами используется memory mapping.)
# Инструкция по сборке и запуску:
```sh
mkdir build
cd build
cmake ../CMakelists.txt
make
./main
```
