# Задание.
Задаётся радиус окружности. Необходимо с помощью метода Монте-Карло рассчитать её 
площадь.
# Инструкция по сборке и запуску:
```sh
mkdir build
cd build
cmake ../CMakelists.txt
make
./lab_exe <radius> <number of threads> <number of points (increses accuracy)>
```
